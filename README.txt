A clean, easy way to display a latest tweets widget on your WordPress powered site.

== Description ==

This plugin adds a reusable, customizable widget in Appearance > Widgets. Set the title, maximum and Twitter username on a per-widget basis. CSS3 enhanced to display tweets in a speech bubble.

== Installation ==

1. Upload the plugin in 'Plugins > Add New' or FTP into the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Appearance > Widgets
4. Drag and drop the widget into your sidebar
5. Adjust the settings and click save

== Frequently Asked Questions ==

= Will the widget fit on my site? =

Yes, it does not have a set width. It fills the space it is given, ideally 150px-300px wide.

= Is this plugin localized / internationalized? =

Yes, a .po file has been generated and included in the lang folder.

= Do I need to modify any theme files to use this plugin? =

No, after the plugin has been activated, you simply drag and drop it into your sidebar.

= What if I only want the widget on page X? =

Try adding a plugin like Widget Logic or Widget context to add more widget control to your site.
